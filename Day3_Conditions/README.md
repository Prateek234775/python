# Day 3 – Operators & Conditional Statements

## Operators
- Arithmetic: + - * / %
- Comparison: == != > < >= <=
- Logical: and, or, not

## Conditional Statements
- if condition is true → block runs
- else runs when condition is false

## Example
if age >= 18:
    print("Adult")
else:
    print("Minor")

## Key Points
- Indentation is mandatory
- Conditions return True or False
