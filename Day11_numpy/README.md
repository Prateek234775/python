# 🧠 What is NumPy?

NumPy stands for Numerical Python.

It is used for:
- Fast calculations
- Arrays
- Matrix operations
- Scientific computing

---

# ⚡ Why NumPy?

Compared to Python lists:

| Python Lists | NumPy Arrays |
|---|---|
| Slower | Faster |
| More memory | Optimized |
| Basic operations | Advanced operations |

---

# 📦 Creating Arrays

```python
import numpy as np

arr = np.array([1, 2, 3, 4])
```

---

# ➕ Array Operations

```python
print(arr + 2)
print(arr * 2)
```

---

# 📐 Array Properties

## Shape
```python
print(arr.shape)
```

## Data Type
```python
print(arr.dtype)
```

---

# 🧱 2D Arrays

```python
arr2 = np.array([
    [1, 2],
    [3, 4]
])
```
