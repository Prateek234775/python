# Day 5 – Functions in Python

## What is a function?
A function is a reusable block of code.

## Syntax
def function_name(parameters):
    statements
    return value

## Types of functions
- Built-in (print, input, len)
- User-defined

## return keyword
Used to send value back from function.

## Advantages
- Code reuse
- Easy debugging
- Clean code
