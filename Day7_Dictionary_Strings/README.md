# Day 7 – Dictionaries & Strings

## Dictionary
- Stores data in key-value form
- Mutable
- Keys must be unique

Example:
student = {
    "name": "Prateek",
    "age": 20
}

## Dictionary Methods
- get()
- keys()
- values()
- items()

## Strings
- Sequence of characters
- Immutable

## Common String Functions
- upper()
- lower()
- replace()
- len()

## Key Points
- Dictionaries are very fast
- Strings cannot be modified directly
