# Day 6 – Lists, Tuples & Sets

## Lists
- Ordered collection
- Mutable
- Allows duplicate values

Example:
numbers = [1, 2, 3]

## Tuples
- Ordered collection
- Immutable

Example:
data = (1, 2, 3)

## Sets
- Unordered collection
- No duplicate values

Example:
s = {1, 2, 3}

## Key Difference
List → mutable i
Tuple → immutable
Set → unique values only
