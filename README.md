# What is Seaborn?

Seaborn is a Python library for:

- Statistical graphics
- Better visualizations
- Professional charts

Built on top of Matplotlib.

---

# Visualizations Learned

## Count Plot

```python
sns.countplot()
```

Used for category counts.

---

## Histogram

```python
sns.histplot()
```

Shows distribution of values.

---

## Box Plot

```python
sns.boxplot()
```

Used to detect:
- Outliers
- Spread of data

---

## Heatmap

```python
sns.heatmap()
```

Used to visualize correlations.
