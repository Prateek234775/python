# 📊 What is Matplotlib?

Matplotlib is a Python library used for:
- graphs
- charts
- visualization

Used heavily in:
- Data Science
- AI/ML
- Analytics

---

# 📈 Line Graph

```python
plt.plot(x, y)
```

Used for:
- trends
- growth
- changes over time

---

# 📊 Bar Chart

```python
plt.bar(x, y)
```

Used for:
- comparisons
- categories

---

# 🥧 Pie Chart

```python
plt.pie(values)
```

Used for:
- percentages
- distributions

---

# 🛠️ Functions Learned

| Function | Purpose |
|---|---|
| plt.plot() | Line graph |
| plt.bar() | Bar chart |
| plt.pie() | Pie chart |
| plt.title() | Chart title |
| plt.xlabel() | X-axis label |
| plt.ylabel() | Y-axis label |
| plt.show() | Display graph |

# 🚀 Why Visualization Matters

Visualization helps:
